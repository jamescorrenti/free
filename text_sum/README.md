## To Run:
```
$ pip3 install -r requirements
$ python3 install.py
$ sudo python3 summary.py
```
Then browse to http://localhost

## Python Dependencies
* nltk

## Natural Language Tool Kit Dependencies:
* punkt
* stopwords
* averaged_perception_tagger
* wordnet
